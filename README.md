## 极简 Markdown PPT

基于[reveal.js](https://github.com/hakimel/reveal.js)开发的支持markdown语法对PPT进行快速编写，让排版变的更轻松

### 使用方法：
鼠标放在页面上方即可显示设置按钮，对PPT进行编写
### 编写规则:
#表示首页 ##表示主页面 ###表示子页面 , 按方向键可进行页面切换，左右键进行主页面切换，上下键进行当前主页面的子页面切换。
```
# 1 slide //首页
## 2 slide //第二页
## 3 slide //第三页
## 4 slide //第四页
### 4.1 slide //第四页的子页面
### 4.2 slide //第四页的子页面
```

### 功能：
支持主题切换,pdf打印,演讲者模式等等。
