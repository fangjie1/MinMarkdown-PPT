## 极简 Markdown PPT

基于[reveal.js](https://github.com/hakimel/reveal.js)开发的支持markdown语法对PPT进行快速编写，让排版变的更轻松

### 使用方法：
鼠标放在页面上方即可显示设置按钮，对PPT进行编写
### 编写规则:
#表示首页 ##表示主页面 ###表示子页面 , 按方向键可进行页面切换
```
# one slide
## two slide
## three slide 
## four slide 
### four slide
### four slide
```

### 功能：
支持主题切换,pdf打印,演讲者模式等等。
